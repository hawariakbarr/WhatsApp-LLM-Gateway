# WhatsApp LLM Gateway

🚀 A complete WhatsApp Web API integration with multiple LLM providers (Claude, GPT-4, Gemini).

## Features

- ✅ WhatsApp Web integration via Baileys
- ✅ Multi-LLM support (Anthropic, OpenAI, Google)
- ✅ Automatic model routing & fallback
- ✅ API rate limit monitoring
- ✅ Indonesian language detection
- ✅ Parenting context awareness
- ✅ Session persistence
- ✅ REST API for management
